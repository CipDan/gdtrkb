STYLE DIRECTIONS · v1

Three takes on “retro accent, modern bones”

Same shared frame in all three: a retro display face used only for the wordmark, headings, and the score column; a clean sans (or legible mono) for everything you actually read; chunky bordered cards and buttons; the results table as a high-score board.

What changes is which corner of the retro-games world the flavor comes from — and how much execution each one asks for. Every panel shows the same pieces (wordmark, palette, type, a tool “cartridge” card, badges, buttons, search, high-score rows) using your real seed data, so you're comparing like for like.

Wordmark “TOOLDEX” is a placeholder (tool + -dex, like a catalog/Pokédex). Swap for the real name later — it's here only to show how the display face sets.
